# Phase 01 Linear Structures Mission

## Learning goals

- Understand sequential data organization and access patterns
- Learn append, insert, delete, and traversal tradeoffs
- Practice object-oriented design with simple classes
- Connect linear data structures to practical systems behavior

## Manual coding boundaries

The learner should manually implement the primary data structures in this phase.

## Structures to implement manually

- Dynamic Array
- Singly Linked List
- Doubly Linked List
- Stack
- Queue
- Deque

## Experiment target

Build one stack-based and one queue-based mini system after the structures are ready.

## Exit criteria

- All Phase 1 structure tests pass
- You can explain complexity tradeoffs for each structure
- You can describe when one structure is preferable to another
