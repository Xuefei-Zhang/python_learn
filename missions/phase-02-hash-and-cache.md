# Phase 02 Hash and Cache Mission

## Learning goals

- Understand hashing, equality, collisions, and resizing
- Learn why Python dictionaries and sets are powerful
- Connect lookup-heavy structures to practical caching systems

## Manual coding boundaries

The learner should manually implement the primary hash-based structures in this phase.

## Structures to implement manually

- Hash Map
- Hash Set
- LRU Cache

## Experiment target

Build a simple cache service and a lightweight log indexer.

## Exit criteria

- All Phase 2 tests pass
- You can explain hashing tradeoffs and collision handling
- You can justify the design of an LRU cache
