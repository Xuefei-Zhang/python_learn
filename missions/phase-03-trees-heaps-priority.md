# Phase 03 Trees Heaps Priority Mission

## Learning goals

- Understand hierarchical and ordered structures
- Practice recursive and iterative reasoning
- Use trees and heaps in realistic tasks

## Manual coding boundaries

The learner should manually implement the main structures and supporting logic.

## Structures to implement manually

- Binary Search Tree
- Heap
- Priority Queue
- Trie

## Experiment target

Build autocomplete and top-k tracking experiments.

## Exit criteria

- All Phase 3 tests pass
- You can explain heap and tree use cases clearly
- You can compare ordered and unordered retrieval strategies
