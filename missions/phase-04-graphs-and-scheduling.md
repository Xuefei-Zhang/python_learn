# Phase 04 Graphs and Scheduling Mission

## Learning goals

- Understand graph modeling and traversal
- Learn dependency ordering and path-finding basics
- Apply graph algorithms to practical scheduling scenarios

## Manual coding boundaries

The learner should manually implement graph structures and core algorithms.

## Structures and algorithms to implement manually

- Graph
- BFS
- DFS
- Topological Sort
- Dijkstra

## Experiment target

Build a dependency scheduler and path-finding experiment.

## Exit criteria

- All Phase 4 tests pass
- You can explain traversal and shortest-path tradeoffs
- You can reason about dependency graphs in engineering systems
